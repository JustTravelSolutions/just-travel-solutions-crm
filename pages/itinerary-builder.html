Intuitive Drag-and-Drop Editor

Create professional itineraries in minutes

  • Visual Editor: Build itineraries visually with no coding required
  • Pre-designed Blocks: Choose from dozens of content blocks for flights, hotels, activities, and more
  • Custom Templates: Start with professionally designed templates or create your own
  • Day-by-Day Planning: Organize travel plans by day with visual timeline
  • Rich Media Support: Add photos, videos, maps, and interactive elements
  • Branding Options: Customize with your agency's logo, colors, and fonts

Powerful Content Import

Save time by importing content directly from suppliers

  • Airline Import: Import flight details including schedules, aircraft, and amenities
  • Cruise Line Import: Import cruise itineraries, ship details, and cabin information
  • Tour Operator Import: Import tour packages with day-by-day breakdowns
  • Hotel Import: Import hotel descriptions, room types, and amenities
  • URL Pasting: Import content by simply pasting a supplier website URL
  • Image Import: Automatically import high-quality images from suppliers

Smart Content Library

Build a reusable library of your best content

  • Tagging System: Organize content with customizable tags for easy retrieval
  • Smart Suggestions: Automatic tag suggestions based on content
  • Advanced Search: Find content quickly with powerful search capabilities
  • Content Categories: Organize by supplier type, destination, or custom categories
  • Usage Tracking: See which content is used most frequently
  • Content Updates: Update saved items once to reflect changes across all itineraries

Interactive Client Experience

Engage your clients with interactive itineraries

  • Interactive Maps: Show the full journey with interactive maps
  • Day-by-Day Navigation: Clients can easily navigate through their travel days
  • Weather Forecasts: Integrated weather information for travel dates
  • Booking Confirmation: Allow clients to confirm bookings directly from the itinerary
  • Client Notes: Enable clients to add their own notes to the itinerary
  • Mobile App Access: Optional companion app for clients to access itineraries offline
